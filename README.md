  ## About the repo
This mini-project is an image processing tool that applies basic operations (filters, equalization binarization,..) on PGM images.

  ## Technologies
  * Python 3
  * Tkinter: Python GUI library
  * Numpy
  * Matplotlib

 ## Run the app:
```
python window.py
```
